# Labs for L200 - KQL Advanced Kusto

Below you will find the labs for the KQL Advanced Kusto course (L200).  These should be distributed out during the course, but generally follow the order listed below.  Ensure to follow the prerequisites for each of the labs before proceeding into the steps of each lab.  These labs do require you to have access to a Kusto Query Cluster and an Azure Subscription to house resources for the data sets.
